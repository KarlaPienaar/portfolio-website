# Portfolio Website

## Overview

This repository contains the source code for my personal portfolio website, developed as part of my BSc Information Technology Profile Development assignment at North-West University.

The website serves as my virtual CV and showcases my education, skills, experience, projects, and contact information.

## Live Website

https://karlapienaar.github.io/portfolio-website/

## Features

- Home page
- About Me
- Portfolio
- Projects
- Contact page
- Downloadable CV
- Links to GitHub and LinkedIn

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Project Structure


portfolio-website/
│ index.html
│ aboutpage.html
│ portfoliopage.html
│ projectspage.html
│ contactpage.html
│ style.css
│ script.js
│ images/
│ CV.pdf


## Running the Project

1. Clone the repository.
2. Open the project in Visual Studio Code.
3. Open `index.html` using Live Server.

Alternatively, visit the live website:

https://karlapienaar.github.io/portfolio-website/
